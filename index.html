How to Contribute (Git)

Recommended workflow (summarized):

  1. Fork the repo, clone locally: git clone <your-fork-url>
  2. Create a branch: git checkout -b feat/add-new-resources
  3. Edit index.html, stage & commit: git add . && git commit -m "feat: add resource X"
  4. Push & open a pull request: git push -u origin feat/add-new-resources

Keep messages short & conventional (feat, fix, docs, style, chore).
